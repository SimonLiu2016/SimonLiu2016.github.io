1. Introduction

KeyMaster is a shortcut management tool designed specifically for macOS, helping users more efficiently manage and use system and application shortcuts. With KeyMaster, you can:

  • View all shortcuts for the current application
  • Download and use application shortcut packages
  • Quickly search and use shortcuts
  • Use the floating window feature for quick access to shortcuts

2. Installation and Launch

System Requirements

  • macOS 10.15 (Catalina) or later
  • At least 200MB of available disk space

Installation Steps

  1. Download and install KeyMaster from the macOS App Store
  2. On first launch, the system will prompt you to grant necessary permissions

3. Main Features

3.1 Automatic Shortcut Capture

KeyMaster can automatically identify and display all available shortcuts for the current application, without requiring manual input or search.

3.2 Application Shortcut Packages

The app integrates application shortcut package functionality. The free version supports shortcuts for macOS built-in and system applications. The paid version allows downloading of all shortcut packages. Shortcut packages are regularly updated and new software support is added.

3.3 Floating Window Feature

Through the floating window feature, you can quickly access and use shortcuts, improving work efficiency.

3.4 Multi-language Support

The app supports the top 10 global languages, including Chinese, English, Japanese, Korean, French, German, Spanish, Russian, Arabic, and Portuguese.

5. Usage Methods

6.1 Finding Shortcuts

  1. Enter keywords in the search box on the main interface
  2. Results will display matching shortcuts and their descriptions

6.2 Using Shortcuts

  1. Find the required shortcut in the results list
  2. Press the shortcut combination directly on the keyboard

6.3 Downloading Shortcut Packages

The free version supports shortcuts for macOS built-in and system applications. The paid version allows downloading of all shortcut packages. During your subscription, you can download all application shortcut packages. After subscription expires, third-party application shortcut packages cannot be downloaded, but already downloaded shortcut packages can continue to be used normally.

7. Troubleshooting

7.1 App Won't Launch

  • Check if macOS version meets minimum requirements
  • Confirm that necessary system permissions have been granted
  • Try reinstalling the app

7.2 Shortcuts Not Recognized

  • Confirm that Accessibility permission has been granted
  • Check if the target application supports shortcut capture
  • Try manually refreshing the shortcut list

7.3 Subscription Issues

  • Check if subscription status is normal
  • Confirm network connection is normal
  • Contact technical support if needed
