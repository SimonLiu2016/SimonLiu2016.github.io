1. 介绍

KeyMaster 是一款专为 macOS 设计的快捷键管理工具，帮助用户更高效地管理和使用系统及应用快捷键。通过 KeyMaster，您可以：

  • 查看当前应用的所有快捷键
  • 下载和使用应用快捷键包
  • 快速搜索和使用快捷键
  • 使用悬浮窗功能快速访问快捷键

2. 安装与启动

系统要求

  • macOS 10.15 (Catalina) 或更高版本
  • 至少 200MB 可用磁盘空间

安装步骤

  1. 从 macOS App Store 下载安装 KeyMaster
  2. 首次启动时，系统会提示您授予必要的权限

3. 主要功能

3.1 快捷键自动抓取

KeyMaster 能够自动识别并显示当前应用的所有可用快捷键，无需手动输入或查找。

3.2 应用快捷键包

本应用集成了应用快捷键包功能。免费版本支持 macOS 内置及系统应用的快捷键，付费版本可以下载全部快捷键包。快捷键包会定期更新并增加新的软件支持。

3.3 悬浮窗功能

通过悬浮窗功能，您可以快速访问和使用快捷键，提高工作效率。

3.4 多语言支持

本应用支持全球 10 大语言，包括中文、英语、日语、韩语、法语、德语、西班牙语、俄语、阿拉伯语和葡萄牙语。

4. 界面说明

4.1 主界面布局

  • 顶部导航栏：包含搜索框、功能按钮等
  • 左侧侧边栏：分类导航和应用列表
  • 中间内容区：快捷键列表和详细信息
  • 底部状态栏：当前状态和提示信息

4.2 快捷键列表

快捷键列表显示以下信息：

  • 快捷键组合（如 ⌘+S）
  • 功能描述
  • 使用频率（如果启用了历史记录）
  • 操作类型（编辑、导航、视图等）

5. 使用方法

5.1 查找快捷键

  1. 在主界面的搜索框中输入关键词
  2. 结果将显示匹配的快捷键及其说明

5.2 使用快捷键

  1. 在结果列表中找到所需的快捷键
  2. 直接在键盘上按下快捷键组合

5.3 下载快捷键包

免费版本支持 macOS 内置及系统应用的快捷键。付费版本可以下载全部快捷键包。订阅期间，您可以下载全部与应用快捷键包，订阅过期后，第三方应用快捷键包不可下载，但已经下载的第三方应用快捷键包可以正常使用。

6. 设置选项

6.1 通用设置

  • 主题：选择浅色或深色主题
  • 语言：选择界面显示语言

6.2 订阅设置

  • 订阅状态：查看当前订阅状态
  • 订阅管理：管理订阅计划
  • 高级功能：查看可用的高级功能

7. 故障排除

7.1 应用无法启动

  • 检查 macOS 版本是否满足最低要求
  • 确认已授予必要的系统权限
  • 尝试重新安装应用

7.2 快捷键无法识别

  • 确认已授予辅助功能权限
  • 检查目标应用是否支持快捷键抓取
  • 尝试手动刷新快捷键列表

7.3 性能问题

  • 关闭不必要的功能（如 AI 搜索）
  • 定期清理快捷键历史记录
  • 检查系统资源使用情况
