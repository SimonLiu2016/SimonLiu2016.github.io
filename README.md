# SimonLiu2016.github.io

This is a personal website built with GitHub Pages. The site features:

- Modern software company design
- Internationalization support (English and Chinese)
- Responsive layout for all devices
- Multiple pages including Home, About, Projects, and Contact
- Open source project showcase
- Relative paths for GitHub Pages compatibility

## Pages

- [Home](index.html)
- [About](about.html)
- [Projects](projects.html)
- [Contact](contact.html)

## Features

- Clean, modern design with CSS Grid and Flexbox
- Interactive elements with JavaScript
- Mobile-friendly responsive design
- Language switching functionality
- Project filtering on the Projects page
- Contact form with validation

## Deployment

This site is automatically deployed to GitHub Pages from the main branch.
